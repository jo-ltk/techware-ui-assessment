# Techware UI Assessment

A responsive Next.js application built based on the provided Figma design, including the required scroll-based interactions and animations.

## Getting Started

### Install dependencies

```bash
npm install
```

### Start the development server

```bash
npm run dev
```

Open:

http://localhost:3000

### Other commands

```bash
npm run build
npm run start
npm run lint
```

> Node.js 20 or later is required.
